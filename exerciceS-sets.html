EXERCICE 1 :

Écrire un programme qui demande à l'utilisateur de saisir deux ensembles d'entiers, puis effectue les opérations suivantes :
  • Affiche l'ensemble union des deux ensembles.
  • Affiche l'ensemble intersection des deux ensembles.
  • Affiche l'ensemble différence entre le premier et le deuxième ensemble.
  • Affiche l'ensemble différence symétrique entre les deux ensembles.
EXEMPLE D'EXECUTION :
Saisissez le premier ensemble d'entiers (séparés par des espaces) : 1 2 3 4
Saisissez le deuxième ensemble d'entiers (séparés par des espaces) : 3 4 5 6
Union des deux ensembles : {1, 2, 3, 4, 5, 6}
Intersection des deux ensembles : {3, 4}
Différence entre le premier et le deuxième ensemble : {1, 2} Différence symétrique entre les deux ensembles : {1, 2, 5, 6}
SOLUTION :
# Demande à l'utilisateur de saisir deux ensembles d'entiers
set1_str = input("Saisissez le premier ensemble d'entiers (séparés par des espaces) : ")
set2_str = input("Saisissez le deuxième ensemble d'entiers (séparés par des espaces) : ")
set1_list = [int(x) for x in set1_str.split()]
set2_list = [int(x) for x in set2_str.split()]
# Convertit les listes en ensembles
set1 = set(set1_list)
set2 = set(set2_list)
# Affiche l'ensemble union
union = set1.union(set2)
print("Union des deux ensembles :", union)
# Affiche l'ensemble intersection
intersection = set1.intersection(set2)
print("Intersection des deux ensembles :", intersection)
# Affiche l'ensemble différence (premier ensemble - deuxième ensemble)
difference = set1.difference(set2)
print("Différence entre le premier et le deuxième ensemble :", difference)
# Affiche l'ensemble différence symétrique
symmetric_difference = set1.symmetric_difference(set2)
print("Différence symétrique entre les deux ensembles :", symmetric_difference)
EXPLICATION :
Tout d'abord,
on demande à l'utilisateur de saisir les ensembles d'entiers sous forme de chaînes de caractères. Les chaînes de caractères sont ensuite divisées en listes d'entiers en utilisant la méthode split() et une compréhension de liste. Ensuite, les listes d'entiers sont converties en ensembles à l'aide de la fonction set(). Les opérations sur les ensembles et l'affichage des résultats restent les mêmes que dans la version précédente.

EXERCICE 4 :

EXEMPLE D'EXECUTION :
Saisissez un ensemble de mots (séparés par des espaces) : chat chien oiseau éléphant
Le mot le plus long de l'ensemble : éléphant

Ensuite,
On utilise la méthode split() pour diviser la chaîne de caractères en une liste de mots.
On utilise la fonction max() avec le paramètre key=len pour trouver le mot le plus long dans la liste de mots. La fonction max() compare les mots en se basant sur leur longueur.
Enfin,
Le mot le plus long est affiché à l'écran.

EXERCICE 5 :

Les listes de mots obtenues sont converties en ensembles à l'aide de la fonction set().
On utilise l'opérateur de différence - pour trouver les mots présents dans le premier ensemble mais pas dans le deuxième.
L'ensemble de différence est converti en une liste à l'aide de la fonction list().
Enfin,
La liste de différence est affichée à l'écran à l'aide de la fonction print().

EXERCICE 10:

Écrire un programme qui demande à l'utilisateur de saisir un ensemble de nombres, puis affiche une nouvelle liste contenant les nombres qui sont multiples de 3.
EXEMPLE D'EXECUTION :
Saisissez un ensemble de nombres (séparés par des espaces) : 10 15 20 25 30 35
Nombres multiples de 3 : [15, 30]
SOLUTION :
# Demande à l'utilisateur de saisir un ensemble de nombres (séparés par des espaces)
nums_input = input("Saisissez un ensemble de nombres (séparés par des espaces) : ")
# Convertit la chaîne de caractères en une liste de nombres
nums_list = [int(num) for num in nums_input.split()]
# Crée une nouvelle liste contenant les nombres multiples de 3
multiples_of_3 = [num for num in nums_list if num % 3 == 0]
# Affiche la liste des nombres multiples de 3
print("Nombres multiples de 3 :", multiples_of_3)
EXPLICATION :
Tout d'abord,
Dans ce programme, nous demandons à l'utilisateur de saisir un ensemble de nombres sous forme de chaîne de caractères. Ensuite, nous utilisons la méthode split() pour diviser cette chaîne en une liste de nombres individuels. Ensuite, nous utilisons une compréhension de liste pour filtrer les nombres qui sont multiples de 3 en utilisant l'opérateur de modulo %. Enfin, nous affichons la liste des nombres multiples de 3.
