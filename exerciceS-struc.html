EXERCICE 1 :

Définir une structure appelée "Personne" contenant les champs suivants : nom, prénom, âge et adresse. Ensuite, créez une variable de type "Personne" et initialisez-la avec des valeurs. Enfin, affichez les informations de la personne.
SOLUTION :
#include <stdio.h>
// Définition de la structure Personne
struct Personne {
char nom[50];
char prenom[50];
int age;
char adresse[100];
};
int main() {
// Création d'une variable de type Personne et initialisation
struct Personne personne1 = {"Doe", "John", 30, "Rue de la Paix, Paris"};
// Affichage des informations de la personne
printf("Nom: %s\n", personne1.nom);
printf("Prénom: %s\n", personne1.prenom);
printf("Âge: %d\n", personne1.age);
printf("Adresse: %s\n", personne1.adresse);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans cet exercice, nous avons défini une structure appelée "Personne" avec les champs nom, prénom, âge et adresse. Ensuite, nous avons créé une variable de type "Personne" appelée "personne1" et initialisé ses valeurs. Enfin, nous avons affiché les informations de la personne à l'aide de la variable "personne1".

EXERCICE 2 :

Énoncé : Définissez une structure appelée "Point" contenant les champs x et y (coordonnées). Ensuite, créez une fonction appelée "distance" qui prend deux points en argument et renvoie la distance entre ces deux points.
SOLUTION :
#include <stdio.h>
#include <math.h>
// Définition de la structure Point
struct Point {
double x;
double y;
};
// Fonction pour calculer la distance entre deux points
double distance(struct Point p1, struct Point p2) {
return sqrt(pow(p2.x - p1.x, 2) + pow(p2.y - p1.y, 2));
}
int main() {
struct Point point1 = {0.0, 0.0};
struct Point point2 = {3.0, 4.0};
double dist = distance(point1, point2);
printf("Distance entre les points : %.2f\n", dist);
return 0;
}
EXPLICATION :
Tout d'abord,
dans cet exercice, nous avons défini une structure "Point" contenant les champs x et y pour représenter les coordonnées d'un point. Ensuite, nous avons créé une fonction "distance" qui prend deux points en argument et utilise la formule de la distance entre deux points dans un plan cartésien pour calculer et renvoyer la distance entre ces deux points.

#include <stdio.h>

struct Rectangle {

};

return rect.longueur * rect.largeur;

EXERCICE 4 :

Définissez une structure appelée "Date" contenant les champs jour, mois et année. Ensuite, créez une fonction appelée "afficherDate" qui prend une date en argument et l'affiche au format "jour/mois/année".

#include <stdio.h>

struct Date {

};

}
int main() {
struct Date date = {5, 9, 2022};
afficherDate(date);
return 0;
}
EXPLICATION :
Tout d'abord,
dans cet exercice, nous avons défini une structure "Date" avec les champs jour, mois et année pour représenter une date. Ensuite, nous avons créé une fonction "afficherDate" qui prend une date en argument et l'affiche en utilisant le format "jour/mois/année". La fonction utilise le spécificateur de format "%02d" pour afficher le jour et le mois avec deux chiffres, en ajoutant un zéro devant si nécessaire.

EXERCICE 7:

Définissir une structure appelée "Employe" contenant les champs nom, salaire et anciennete (en années). Ensuite, créez une fonction appelée "augmenterSalaire" qui prend un employé en argument et augmente son salaire de 10%. Affichez ensuite les informations de l'employé après l'augmentation.
SOLUTION :
#include <stdio.h>
// Définition de la structure Employe
struct Employe {
char nom[50];
float salaire;
int anciennete;
};
// Fonction pour augmenter le salaire d'un employé
void augmenterSalaire(struct Employe *employe) {
employe->salaire *= 1.1;

struct Employe employe1 = {"Durand", 2500.0, 5};
augmenterSalaire(&employe1);
printf("Nom: %s\n", employe1.nom);
printf("Salaire après augmentation: %.2f\n", employe1.salaire);
printf("Ancienneté: %d ans\n", employe1.anciennete);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans cet exercice, nous avons défini une structure "Employe" avec les champs nom, salaire et anciennete pour représenter les informations d'un employé. Ensuite, nous avons créé une fonction "augmenterSalaire" qui prend un pointeur vers un employé en argument et augmente son salaire de 10% en utilisant l'opérateur "->" pour accéder au champ salaire de la structure. Nous avons ensuite appelé cette fonction avec l'adresse de la variable employe1. Enfin, nous avons affiché les informations de l'employé après l'augmentation.

EXERCICE 9:

Définir une structure appelée "Vecteur" contenant les champs x, y et z. Ensuite, créez une fonction appelée "produitScalaire" qui prend deux vecteurs en argument et renvoie leur produit scalaire.
SOLUTION :
#include <stdio.h>
// Définition de la structure Vecteur
struct Vecteur {
float x;
float y;
float z;
};
// Fonction pour calculer le produit scalaire de deux vecteurs
float produitScalaire(struct Vecteur v1, struct Vecteur v2) {
return v1.x * v2.x + v1.y * v2.y + v1.z * v2.z;
}
int main() {
struct Vecteur vecteur1 = {2.0, -1.5, 3.7};
struct Vecteur vecteur2 = {1.5, 4.2, -2.3};
float produit = produitScalaire(vecteur1, vecteur2);
printf("Produit scalaire : %.2f\n", produit);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans cet exercice, nous avons défini une structure "Vecteur" avec les champs x, y et z pour représenter les coordonnées d'un vecteur. Ensuite, nous avons créé une fonction "produitScalaire" qui prend deux vecteurs en argument et renvoie leur produit scalaire en utilisant la formule mathématique du produit scalaire. Nous avons ensuite appelé cette fonction avec les vecteurs vecteur1 et vecteur2 et affiché le résultat du produit scalaire.

EXERCICE 10:

Définir une structure appelée "PointeurVersFonction" contenant un pointeur vers une fonction qui prend deux entiers en argument et renvoie leur somme. Ensuite, créez une fonction appelée "addition" qui prend deux entiers en argument et renvoie leur somme. Créez une variable de type "PointeurVersFonction" et assignez-lui le pointeur vers la fonction "addition". Utilisez ensuite le pointeur pour appeler la fonction et afficher le résultat.
SOLUTION :
#include <stdio.h>
// Définition de la structure PointeurVersFonction
struct PointeurVersFonction {
int (*fonction)(int, int);
};
// Fonction addition
int addition(int a, int b) {
return a + b;
}
int main() {
struct PointeurVersFonction pointeur;
// Assignation du pointeur vers la fonction addition
pointeur.fonction = addition;
int resultat = pointeur.fonction(5, 3);
printf("Résultat : %d\n", resultat);
return 0;
}
EXPLICATION :
Tout d'abord,
cDans cet exercice, nous avons défini une structure "PointeurVersFonction" avec un champ "fonction" qui est un pointeur vers une fonction prenant deux entiers en argument et renvoyant un entier. Ensuite, nous avons créé une fonction "addition" qui prend deux entiers en argument et renvoie leur somme. Nous avons créé une variable de type "PointeurVersFonction" et assigné le pointeur vers la fonction "addition" à son champ "fonction". En utilisant le pointeur, nous avons appelé la fonction et affiché le résultat.
