EXERCICE 1 :

Écrire une fonction qui prend deux entiers en entrée et retourne leur somme.
SOLUTION :
#include <stdio.h>
int somme(int a, int b) {
return a + b;
} int main() {
int x, y;
printf("Saisissez deux entiers : ");
scanf("%d %d", &x, &y);
int resultat = somme(x, y);
printf("La somme est : %d\n", resultat);
return 0;
}
EXPLICATION :
Tout d'abord,
La fonction somme prend deux entiers a et b en entrée, les additionne et retourne le résultat.

EXERCICE 3 :

Écrire une fonction qui prend trois nombres réels en entrée et retourne leur moyenne.
SOLUTION :
#include <stdio.h>
float calculerMoyenne(float a, float b, float c) {
return (a + b + c) / 3;
}
int main() {
float x, y, z;
printf("Saisissez trois nombres réels : ");
scanf("%f %f %f", &x, &y, &z);
float resultat = calculerMoyenne(x, y, z);
printf("La moyenne est : %.2f\n", resultat);
return 0;
}
EXPLICATION :
Tout d'abord,
La fonction calculerMoyenne prend trois nombres réels a, b et c en entrée, les additionne et divise le résultat par 3 pour obtenir la moyenne.

EXERCICE 4 :

Écrire une fonction qui prend une année en entrée et vérifie si elle est bissextile.
SOLUTION :
#include <stdio.h>
int estBissextile(int annee) {
if ((annee % 4 == 0 && annee % 100 != 0) || annee % 400 == 0) {
return 1;
}
else {
return 0;
}
}
int main() {
int an;
printf("Saisissez une année : ");
scanf("%d", &an);
if (estBissextile(an)) {
printf("%d est une année bissextile.\n", an);
}
else {
printf("%d n'est pas une année bissextile.\n", an);
}
return 0;
}
EXPLICATION :
Tout d'abord,
La fonction estBissextile prend une année annee en entrée et vérifie si elle est bissextile en appliquant les règles du calendrier grégorien.

EXERCICE 6:

Écrire une fonction qui prend un entier en entrée et retourne sa factorielle.
SOLUTION :
#include <stdio.h>
int calculerFactorielle(int n) {
int resultat = 1;
for (int i = 1; i <= n; i++) {
resultat *= i;
}
return resultat;
}
int main() {
int x;
printf("Saisissez un entier : ");
scanf("%d", &x);
int resultat = calculerFactorielle(x);
printf("La factorielle de %d est : %d\n", x, resultat);
return 0;
}
EXPLICATION :
Tout d'abord,
La fonction calculerFactorielle prend un entier n en entrée et calcule sa factorielle en utilisant une boucle for pour multiplier tous les entiers de 1 à n.

EXERCICE 7:

Écrire une fonction qui prend un entier en entrée et retourne le nombre de chiffres qu'il contient.
SOLUTION :
#include <stdio.h>
int compterChiffres(int nombre) {
int count = 0;
while (nombre != 0) {
nombre /= 10;
count++;
}
return count;
}
int main() {
int n;
printf("Saisissez un entier : ");
scanf("%d", &n);
int chiffres = compterChiffres(n);
printf("Le nombre de chiffres est : %d\n", chiffres);
return 0;
}
EXPLICATION :
Tout d'abord,
la fonction compterChiffres prend un entier nombre en entrée et divise répétitivement le nombre par 10 jusqu'à ce qu'il devienne 0, en incrémentant un compteur à chaque division.

EXERCICE 9:

Écrivez un programme en C qui convertit une température donnée en degrés Celsius en degrés Fahrenheit. Le programme demande à l'utilisateur d'entrer une température en Celsius, puis utilise une fonction celsiusVersFahrenheit pour effectuer la conversion et renvoyer la valeur en Fahrenheit. Enfin, le résultat est affiché à l'utilisateur.
SOLUTION :
#include <stdio.h>
float celsiusVersFahrenheit(float celsius) {
return (celsius * 9 / 5) + 32;
}
int main() {
float celsius;
printf("Entrez la température en degrés Celsius : ");
scanf("%f", &celsius);
float fahrenheit = celsiusVersFahrenheit(celsius);
printf("La température en degrés Fahrenheit est : %.2f\n", fahrenheit);
return 0;
}
EXPLICATION :
Tout d'abord,
La fonction celsiusVersFahrenheit prend en paramètre un nombre réel celsius représentant la température en degrés Celsius. Elle effectue la conversion en utilisant la formule : Fahrenheit = (Celsius * 9 / 5) + 32. La fonction retourne le résultat de la conversion en Fahrenheit.

EXERCICE 10:

écrire un programme en C qui calcule le carré d'un nombre entier saisi par l'utilisateur.

Le programme demande à l'utilisateur d'entrer un nombre, puis utilise une fonction carre pour calculer le carré de ce nombre.

Enfin, le résultat est affiché à l'utilisateur.
SOLUTION :
#include <stdio.h>
int carre(int nombre) {
return nombre * nombre;
}
int main() {
int nombre;
printf("Entrez un nombre : ");
scanf("%d", &nombre);
int resultat = carre(nombre);
printf("Le carré de %d est : %d\n", nombre, resultat);
return 0;
}
EXPLICATION :
Tout d'abord,
La fonction carre prend en paramètre un nombre entier nombre et retourne le carré de ce nombre en multipliant le nombre par lui-même.
