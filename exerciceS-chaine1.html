Chaine de caractères

EXERCICE 1 :

Écrivez un programme en C qui demande à l'utilisateur de saisir une chaîne de caractères, puis affiche le nombre de caractères présents dans cette chaîne.
SOLUTION :
#include <stdio.h>
int main() {
char chaine[100];
int nombre_caracteres = 0;
printf("Saisissez une chaîne de caractères : ");
scanf("%s", chaine);
// Calcul du nombre de caractères
for (int i = 0; chaine[i] != '\0'; i++) {
nombre_caracteres++;
}
printf("Le nombre de caractères dans la chaîne est : %d\n", nombre_caracteres);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous utilisons la fonction scanf avec le format %s pour permettre à l'utilisateur de saisir une chaîne de caractères. Ensuite, nous utilisons une boucle for pour parcourir chaque caractère de la chaîne jusqu'à ce que nous atteignons le caractère de fin de chaîne '\0'. À chaque itération, nous incrémentons la variable nombre_caracteres pour compter le nombre de caractères. Enfin, nous affichons le résultat à l'utilisateur.

EXERCICE 2 :

Écrivez un programme en C qui demande à l'utilisateur de saisir une chaîne de caractères, puis affiche cette chaîne inversée. .
SOLUTION :
#include <stdio.h>
#include
int main() {
char chaine[100];
int longueur;
printf("Saisissez une chaîne de caractères : ");
scanf("%s", chaine);
longueur = strlen(chaine);
printf("La chaîne inversée est : ");
for (int i = longueur - 1; i >= 0; i--) {
printf("%c", chaine[i]);
}
printf("\n");
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous utilisons scanf pour permettre à l'utilisateur de saisir une chaîne de caractères. Nous utilisons ensuite strlen pour obtenir la longueur de la chaîne saisie. Ensuite, nous utilisons une boucle for pour parcourir la chaîne de caractères de la fin vers le début, puis nous affichons chaque caractère. Ainsi, la chaîne sera affichée inversée.

Veuillez noter que cette approche n'est pas recommandée car scanf peut provoquer des dépassements de tampon si la chaîne saisie dépasse la taille déclarée du tableau chaine. L'utilisation de fgets est préférable pour éviter ce problème.

EXERCICE 3 :

Écrivez un programme en C qui demande à l'utilisateur de saisir une chaîne de caractères, puis compte le nombre de voyelles présentes dans cette chaîne.
SOLUTION :

#include <ctype.h>
int main() {
char chaine[100];

voyelles++;
}
}
printf("Le nombre de voyelles dans la chaîne est : %d\n", voyelles);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous utilisons scanf pour permettre à l'utilisateur de saisir une chaîne de caractères. Ensuite, nous utilisons une boucle for pour parcourir chaque caractère de la chaîne. À l'intérieur de la boucle, nous utilisons la fonction tolower de la bibliothèque ctype.h pour convertir le caractère en minuscule. Ensuite, nous vérifions si le caractère est une voyelle (a, e, i, o, u) et si c'est le cas, nous incrémentons le compteur de voyelles.

EXERCICE 7:

Écrivez un programme en C qui demande à l'utilisateur de saisir une chaîne de caractères, puis vérifie si cette chaîne est un palindrome en ignorant la casse (majuscules/minuscules).
SOLUTION :
#include <stdio.h>
#include <stdbool.h>
#define MAX_LENGTH 100
int main() {
char chaine[MAX_LENGTH];
int index = 0;
char caractere;
printf("Saisissez une chaîne de caractères : ");
// Lire un caractère à la fois jusqu'à atteindre la fin de la ligne ou la taille maximale
while ((caractere = getchar()) != '\n' && index < MAX_LENGTH) {
chaine[index] = caractere;
index++;
}
chaine[index] = '\0';
// Vérifier si la chaîne est un palindrome en ignorant la casse
bool estPalindrome = true;
int debut = 0;
int fin = index - 1;
while (debut < fin) {
char debutMinuscule = chaine[debut];
char finMinuscule = chaine[fin];
// Convertir les caractères en minuscules si ce sont des lettres
if (debutMinuscule >= 'A' && debutMinuscule <= 'Z') {
debutMinuscule = debutMinuscule + 32;
}
if (finMinuscule >= 'A' && finMinuscule <= 'Z') {
finMinuscule = finMinuscule + 32;
}
// Vérifier si les caractères correspondants sont différents
if (debutMinuscule != finMinuscule) {
estPalindrome = false;
break;
}
debut++;
fin--;
}
if (estPalindrome) {
printf("La chaîne est un palindrome.\n");
}
else {
printf("La chaîne n'est pas un palindrome.\n");
}
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous utilisons une boucle while pour lire un caractère à la fois jusqu'à atteindre la fin de la ligne ou la taille maximale définie par MAX_LENGTH. Nous stockons chaque caractère dans le tableau chaine. Lorsque nous atteignons la fin de la ligne ('\n'), nous ajoutons le caractère nul '\0' pour terminer la chaîne.
Ensuite,
nous utilisons une boucle while pour vérifier si la chaîne est un palindrome en ignorant la casse. Nous utilisons deux indices debut et fin pour parcourir la chaîne à partir des extrémités. À chaque itération, nous comparons les caractères correspondants en les convertissant en minuscules si ce sont des lettres. Si les caractères sont différents, nous changeons la variable estPalindrome à false et sortons de la boucle.
Enfin,
nous affichons le résultat en fonction de la valeur de la variable estPalindrome.

Notez que dans cet exemple, nous ne vérifions pas explicitement si la chaîne dépasse la taille maximale MAX_LENGTH. Si la chaîne dépasse cette taille, des problèmes de dépassement de tampon peuvent se produire. Pour une utilisation plus sûre, il est recommandé de vérifier et de gérer les dépassements de tampon de manière appropriée.

EXERCICE 8:

Écrivez un programme qui demande à l'utilisateur de saisir une phrase, puis affiche les mots de cette phrase dans l'ordre inverse.
EXEMPLE:Saisissez une phrase : Hello world, how are you?
Résultat : you? are how world, Hello
SOLUTION :
#include <stdio.h> #define MAX_LENGTH 100
int main() {
char phrase[MAX_LENGTH];
int index = 0;
char caractere;
printf("Saisissez une phrase : ");
// Lire un caractère à la fois jusqu'à atteindre la fin de la ligne ou la taille maximale
while ((caractere = getchar()) != '\n' && index < MAX_LENGTH) {
phrase[index] = caractere;
index++;
}
phrase[index] = '\0';
// Afficher les mots de la phrase dans l'ordre inverse
int debut = index - 1;
int fin = index - 1;
while (debut >= 0) {
// Trouver le début du mot
while (debut >= 0 && phrase[debut] != ' ') {
debut--;
}
// Afficher le mot
for (int i = debut + 1; i <= fin; i++) {
printf("%c", phrase[i]);
}
// Passer à l'espace précédent pour le prochain mot
fin = debut - 1;
debut--;
// Si ce n'est pas le premier mot, ajouter un espace
if (debut >= 0) {
printf(" ");
}
}
printf("\n");
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons une boucle while pour lire un caractère à la fois jusqu'à atteindre la fin de la ligne ou la taille maximale définie par MAX_LENGTH. Nous stockons chaque caractère dans le tableau phrase. Lorsque nous atteignons la fin de la ligne ('\n'), nous ajoutons le caractère nul '\0' pour terminer la phrase.
Ensuite,
nous utilisons une boucle while pour afficher les mots de la phrase dans l'ordre inverse. Nous utilisons deux indices debut et fin pour délimiter chaque mot. Nous commençons par debut et fin égaux à l'indice de fin de la phrase. Puis, nous déplaçons debut vers la gauche jusqu'à trouver un espace, ce qui indique le début d'un mot. Nous affichons ensuite les caractères entre debut + 1 et fin, ce qui correspond au mot actuel. Ensuite, nous mettons à jour fin et debut pour passer au mot précédent. Nous répétons ce processus jusqu'à ce que debut atteigne le début de la phrase.
Enfin,
nous affichons un saut de ligne pour terminer la sortie.

EXERCICE 9:

Écrire un programme qui demande à l'utilisateur de saisir une chaîne de caractères, puis affiche la même chaîne en inversant la casse des lettres (majuscules en minuscules et vice versa).
#include <stdio.h>
int main() {
char chaine[100];
printf("Saisissez une chaine de caracteres : ");
scanf("%s", chaine);
int i = 0;
while (chaine[i] != '\0') {
if (chaine[i] >= 'a' && chaine[i] <= 'z') {
chaine[i] = chaine[i] - 32; // Convertir en majuscule
}
else if (chaine[i] >= 'A' && chaine[i] <= 'Z') {
chaine[i] = chaine[i] + 32; // Convertir en minuscule
}
i++;
}
printf("La chaine avec la casse inverse : %s\n", chaine);
return 0;
}
EXPLICATION :
Tout d'abord,
Ce programme utilise la fonction scanf pour lire la chaîne de caractères saisie par l'utilisateur. Ensuite, il parcourt chaque caractère de la chaîne à l'aide d'une boucle while. Si le caractère est une lettre minuscule, il est converti en majuscule en soustrayant la valeur 32 à son code ASCII. Si le caractère est une lettre majuscule, il est converti en minuscule en ajoutant la valeur 32 à son code ASCII. Les autres caractères (chiffres, espaces, symboles, etc.) restent inchangés.
Ainsi,
la chaîne résultante affichée sera la même que la chaîne saisie par l'utilisateur, mais avec la casse des lettres inversée. Par exemple, si l'utilisateur saisit "Hello World", le programme affichera "hELLO wORLD".

EXERCICE 10:

Écrire un programme qui demande à l'utilisateur de saisir une chaîne de caractères, puis affiche le nombre de mots présents dans cette chaîne.
SOLUTION :
#include <stdio.h>
int main() {
char chaine[100];
int mots = 0;
int i;
printf("Saisissez une chaine de caracteres : ");
scanf("%[^\n]", chaine);
// Parcours de chaque caractere de la chaine
for (i = 0; chaine[i] != '\0'; i++) {
// Si un caractere est un espace, c'est la fin d'un mot
if (chaine[i] == ' ') {
mots++;
}
}
// Ajout du dernier mot (si la chaine n'est pas vide)
if (i > 0) {
mots++;
}
printf("Le nombre de mots dans la chaine : %d\n", mots);
return 0;
}
EXPLICATION :
Tout d'abord,
Ce programme utilise scanf avec le format %[^\n] pour permettre à l'utilisateur de saisir une chaîne de caractères avec des espaces. Ensuite, il parcourt chaque caractère de la chaîne à l'aide d'une boucle for. Si un caractère est un espace, cela indique la fin d'un mot et on incrémente le compteur de mots. Le dernier mot est également comptabilisé en vérifiant si la variable i est supérieure à 0 (c'est-à-dire que la chaine n'est pas vide).

Enfin, le programme affiche le nombre de mots dans la chaîne saisie par l'utilisateur.
