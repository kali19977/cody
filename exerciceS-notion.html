EXERCICE 1 :

Ecrire un programme en Python qui demande à l'utilisateur de saisir deux nombres, puis calcule et affiche leur somme.
EXEMPLE D'EXECUTION :
Saisissez le premier nombre : 3.5
Saisissez le deuxième nombre : 2.1
La somme de 3.5 et 2.1 est : 5.6
SOLUTION :
nombre1 = float(input("Entrez le premier nombre : "))
nombre2 = float(input("Entrez le deuxième nombre : "))
somme = nombre1 + nombre2
print("La somme de", nombre1, "et", nombre2, "est égale à", somme)
EXPLICATION :
Tout d'abord,
le programme utilise la fonction input() pour demander à l'utilisateur de saisir deux nombres. La fonction input() permet de recevoir une saisie de l'utilisateur sous forme de chaîne de caractères. Les nombres saisis par l'utilisateur sont stockés dans les variables nombre1 et nombre2. Notez que nous utilisons float(input(...)) pour convertir la saisie en nombres décimaux (flottants) afin de permettre des calculs plus précis.
Ensuite,
le programme effectue la somme des deux nombres en utilisant l'opérateur d'addition +. Le résultat est stocké dans la variable somme.
Enfin,
le programme utilise la fonction print() pour afficher le résultat de la somme à l'écran. La phrase affichée indique la somme des deux nombres saisis par l'utilisateur.

EXERCICE 5 :

Écrivez un programme python pour saisir le nombre de jours de l'utilisateur et convertissez-le en années, semaines et jours.
EXEMPLE D'EXECUTION :
Entrez le nombre de jours : 456
Le nombre de jours correspond à : 1 années, 15 semaines et 6 jours.
SOLUTION :
jours = int(input("Entrez le nombre de jours : "))
annees = jours // 365
jours_restants = jours % 365
semaines = jours_restants // 7
jours_final = jours_restants % 7
print("Le nombre de jours correspond à :", annees, "années,", semaines, "semaines et", jours_final, "jours.")
EXPLICATION :
Tout d'abord,
Ce programme utilise la fonction input() pour demander à l'utilisateur de saisir un nombre de jours. La valeur saisie est ensuite convertie en nombre entier à l'aide de int().
Ensuite,
le programme effectue les calculs pour convertir les jours en années, semaines et jours.
  • La division entière jours // 365 donne le nombre d'années complètes dans le nombre total de jours.
  • Le reste de la division jours % 365 donne le nombre de jours restants après avoir compté les années complètes.
  • La division entière jours_restants // 7 donne le nombre de semaines complètes dans les jours restants.
  • Le reste de la division jours_restants % 7 donne le nombre de jours restants après avoir compté les semaines complètes.
Enfin,
le programme utilise la fonction print() pour afficher le résultat en indiquant le nombre d'années, de semaines et de jours correspondant.

le programme utilise la fonction print() pour afficher la moyenne des trois entiers à l'écran. Le message "La moyenne des trois entiers est :" est affiché suivi de la valeur de la variable moyenne.

EXERCICE 10:

Ecrivez un programme python pour saisir le côté d'un triangle équilatéral et trouvez la surface du triangle donné.
EXEMPLE D'EXECUTION :
Entrez la longueur du côté du triangle équilatéral : 5
La surface du triangle équilatéral est : 10.825317547305483
SOLUTION :
cote = float(input("Entrez la longueur du côté du triangle équilatéral : "))
surface = (3**0.5 / 4) * cote**2
print("La surface du triangle équilatéral est :", surface)
EXPLICATION :
Tout d'abord,
cote = float(input("Entrez la longueur du côté du triangle équilatéral : ")) : Cette ligne de code demande à l'utilisateur d'entrer la longueur du côté du triangle équilatéral en affichant le message "Entrez la longueur du côté du triangle équilatéral : ". L'entrée de l'utilisateur est capturée à l'aide de la fonction input(), et puis elle est convertie en nombre à virgule flottante (nombre décimal) à l'aide de float() pour être stockée dans la variable cote.
Ensuite,
surface = (3**0.5 / 4) * cote**2 : Cette ligne de code calcule la surface du triangle équilatéral. Dans un triangle équilatéral, la formule pour calculer la surface est : surface = (racine carrée de 3 / 4) * côté^2. Ainsi, on élève 3 à la puissance 0.5 pour obtenir la racine carrée de 3 (3**0.5). On divise ensuite le résultat par 4, et multiplie le tout par le carré de la longueur du côté cote**2. Le résultat est stocké dans la variable surface.
Enfin,
print("La surface du triangle équilatéral est :", surface) : Cette ligne affiche le résultat, c'est-à-dire la surface du triangle équilatéral. Le message "La surface du triangle équilatéral est :" est affiché suivi du résultat du calcul de la surface (stocké dans la variable surface) grâce à la fonction print().
