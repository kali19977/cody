Créer une classe appelée "Rectangle" avec les attributs longueur et largeur, et inclut des méthodes pour calculer l'aire et le périmètre du rectangle

Surface du rectangle 15.0
Périmètre du rectangle : 16.0
SOLUTION :
class Rectangle:
def __init__(self, longueur , largueur):
self.longueur = longueur
self.largueur = largueur
def calculate_area(self):
return self.longueur * self.largueur
def calculate_perimeter(self):
return 2 * (self.longueur + self.largueur)
# User input
longueur = float(input("Entrez la longueur du rectangle : "))
largueur = float(input("Entrez le largueur du rectangle : "))
# Create a Rectangle object
rectangle = Rectangle(longueur , largueur)
# Calculate area and perimeter
area = rectangle.calculate_area()
perimeter = rectangle.calculate_perimeter()
# Print the results
print("Surface du rectangle", area)
print("Périmètre du rectangle :", perimeter)

Ensuite, nous demandons à l'utilisateur d'entrer le nom, l'âge et l'adresse de la personne. Ces valeurs sont utilisées pour créer une nouvelle instance de la classe Personne en utilisant le constructeur.

Nous utilisons ensuite les méthodes de la classe pour afficher les informations de la personne et demander à l'utilisateur de fournir une nouvelle adresse. La méthode modifier_adresse est ensuite appelée pour mettre à jour l'adresse de la personne.
Enfin,
nous utilisons à nouveau la méthode afficher_informations pour afficher les informations mises à jour de la personne. L'utilisation de classes permet d'organiser et de regrouper des données et des comportements liés dans des objets. Cela facilite la manipulation et la gestion des données.
