Chaine de caractères

EXERCICE 1 :

Écrivez un programme qui demande à l'utilisateur de saisir une chaîne de caractères, puis affiche le nombre de caractères présents dans cette chaîne.
Saisissez une chaîne de caractères : Bonjour
Le nombre de caractères dans la chaîne est : 7
SOLUTION :
chaine = input("Saisissez une chaîne de caractères : ")
nombre_caracteres = len(chaine)
print("Le nombre de caractères dans la chaîne est :", nombre_caracteres)
EXPLICATION :
Tout d'abord,
L'utilisateur est invité à saisir une chaîne de caractères à l'aide de la fonction input(). La chaîne saisie est ensuite stockée dans la variable chaine.
Ensuite,
La fonction len() est utilisée pour obtenir la longueur de la chaîne, c'est-à-dire le nombre de caractères qu'elle contient. La valeur retournée par len(chaine) est ensuite assignée à la variable nombre_caracteres.
Enfin,
le programme affiche le nombre de caractères dans la chaîne à l'aide de la fonction print().

EXERCICE 3 :

Écrivez un programme qui demande à l'utilisateur de saisir une chaîne de caractères, puis compte le nombre de voyelles présentes dans cette chaîne.
Saisissez une chaîne de caractères : OpenAI GPT-3
Le nombre de voyelles dans la chaîne est : 4
SOLUTION :
chaine = input("Saisissez une chaîne de caractères : ")
voyelles = "aeiouAEIOU"
compteur_voyelles = 0
for caractere in chaine:
if caractere in voyelles:
compteur_voyelles += 1
print("Le nombre de voyelles dans la chaîne est :", compteur_voyelles)
EXPLICATION :
Tout d'abord,
L'utilisateur est invité à saisir une chaîne de caractères à l'aide de la fonction input(). La chaîne saisie est ensuite stockée dans la variable chaine.
Ensuite,
La variable voyelles est définie comme une chaîne contenant toutes les voyelles en minuscules et en majuscules.
La variable compteur_voyelles est initialisée à 0. Cette variable sera utilisée pour compter le nombre de voyelles présentes dans la chaîne.
Le programme utilise une boucle for pour parcourir chaque caractère dans la chaîne 'chaine'.
À chaque itération de la boucle, le programme vérifie si le caractère actuel est une voyelle en utilisant l'opérateur in pour vérifier si le caractère est présent dans la variable voyelles. Si c'est le cas, le compteur de voyelles compteur_voyelles est incrémenté de 1.
Enfin,
après la boucle, le programme affiche le nombre de voyelles dans la chaîne à l'aide de la fonction print().

EXERCICE 4 :

Écrivez un programme qui demande à l'utilisateur de saisir une chaîne de caractères, puis vérifie si cette chaîne est un palindrome (elle se lit de la même manière de gauche à droite et de droite à gauche).
Saisissez une chaîne de caractères : radar
La chaîne est un palindrome
SOLUTION :
chaine = input("Saisissez une chaîne de caractères : ")
if chaine == chaine[::-1]:
print("La chaîne est un palindrome")
else:
print("La chaîne n'est pas un palindrome")
EXPLICATION :
Tout d'abord,
L'utilisateur est invité à saisir une chaîne de caractères à l'aide de la fonction input(). La chaîne saisie est ensuite stockée dans la variable chaine.
Ensuite,
L'opérateur de découpage de chaîne [::-1] est utilisé pour créer une nouvelle chaîne qui contient tous les caractères de la chaîne d'origine, mais dans l'ordre inverse.
Le programme compare la chaîne d'origine chaine avec sa version inversée chaine[::-1] en utilisant l'opérateur de comparaison ==. Si les deux chaînes sont identiques, cela signifie que la chaîne est un palindrome.
Enfin,
Si la chaîne est un palindrome, le programme affiche "La chaîne est un palindrome". Sinon, il affiche "La chaîne n'est pas un palindrome".

EXERCICE 5 :

Écrivez un programme qui demande à l'utilisateur de saisir une chaîne de caractères, puis remplace toutes les occurrences d'un caractère spécifié par un autre caractère spécifié dans cette chaîne.

Saisissez le caractère à remplacer : o

chaine = input("Saisissez une chaîne de caractères : ")

EXPLICATION :
Tout d'abord,
L'utilisateur est invité à saisir une chaîne de caractères à l'aide de la fonction input(). La chaîne saisie est ensuite stockée dans la variable chaine.
Ensuite,
L'utilisateur est également invité à saisir le caractère à remplacer et le nouveau caractère à utiliser à l'aide de deux autres appels à input(). Ces caractères sont respectivement stockés dans les variables caractere_a_remplacer et nouveau_caractere.
Le programme utilise la méthode replace() sur la chaîne chaine pour remplacer toutes les occurrences du caractère spécifié caractere_a_remplacer par le nouveau caractère spécifié nouveau_caractere. La chaîne modifiée est assignée à la variable chaine_modifiee.
Enfin,
le programme affiche la chaîne modifiée à l'aide de la fonction print().

EXERCICE 8:

Écrivez un programme qui demande à l'utilisateur de saisir une phrase, puis affiche les mots de cette phrase dans l'ordre inverse.
Saisissez une phrase : Je suis en train de programmer
Résultat : programmer de train en suis Je
SOLUTION :
phrase = input("Saisissez une phrase : ")
mots = phrase.split()
mots_inverse = mots[::-1]
phrase_inverse = " ".join(mots_inverse)
print("Résultat :", phrase_inverse)
EXPLICATION :
Tout d'abord,
L'utilisateur est invité à saisir une phrase à l'aide de la fonction input(). La phrase saisie est ensuite stockée dans la variable phrase.
Ensuite,
La méthode split() est utilisée pour diviser la phrase en mots. Par défaut, split() utilise l'espace comme séparateur.
L'opérateur de découpage de liste [::-1] est utilisé pour inverser l'ordre des mots dans la liste mots.
La méthode join() est utilisée pour reconstituer les mots inversés en une seule chaîne de caractères en utilisant l'espace comme séparateur.
Enfin,
le résultat, c'est-à-dire la phrase inversée, est affiché à l'aide de la fonction print().

EXERCICE 9:

Écrire un programme qui demande à l'utilisateur de saisir une chaîne de caractères, puis affiche la même chaîne en inversant la casse des lettres (majuscules en minuscules et vice versa).
Saisissez une chaîne de caractères : ExAmPle
Nouvelle chaîne : eXaMpLE
SOLUTION :
chaine = input("Saisissez une chaîne de caractères : ")
nouvelle_chaine = ""
for caractere in chaine: if caractere.islower():
nouvelle_chaine += caractere.upper()
else:
nouvelle_chaine += caractere.lower()
print("Nouvelle chaîne :", nouvelle_chaine)
EXPLICATION :
Tout d'abord,
Le programme demande à l'utilisateur de saisir une chaîne de caractères. Ensuite, une boucle for est utilisée pour parcourir chaque caractère de la chaîne. Pour chaque caractère, la méthode islower() est utilisée pour vérifier s'il est en minuscule. Si c'est le cas, la méthode upper() est utilisée pour le convertir en majuscule, sinon la méthode lower() est utilisée pour le convertir en minuscule. Le caractère converti est ensuite ajouté à la variable nouvelle_chaine. En fin de compte, la nouvelle chaîne avec la casse inversée est affichée à l'aide de la fonction print().

Ensuite, la fonction len() est utilisée pour obtenir le nombre de mots dans la liste résultante, et ce nombre est affiché à l'aide de la fonction print().
