EXERCICE 1 :

Ecrire un programme en C qui permet de determiner si un entier donné est pair ou impair.
EXEMPLE D'EXECUTION :
Entrez un entier : 17
L'entier 17 est impair.
SOLUTION :
#include <stdio.h>
int main() {
int nombre;
// Saisie de l'entier par l'utilisateur
printf("Entrez un entier : ");
scanf("%d", &nombre);
// Vérification de la parité de l'entier
if (nombre % 2 == 0) {
printf("L'entier %d est pair.\n", nombre);
}
else {
printf("L'entier %d est impair.\n", nombre);
}
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur d'entrer un entier, puis nous le stockons dans la variable nombre. Ensuite, nous vérifions la parité de l'entier en utilisant l'opérateur modulo %. Si le reste de la division de nombre par 2 est égal à 0, cela signifie que l'entier est pair. Sinon, il est impair. Nous affichons ensuite le résultat correspondant à l'aide de la fonction printf().

EXERCICE 2 :

Ecrivez un programme en C pour trouver un maximum entre trois nombres .
EXEMPLE D'EXECUTION :
Entrez le premier nombre : 25
Entrez le deuxième nombre : 10
Entrez le troisième nombre : 32
Le maximum entre les trois nombres est : 32
SOLUTION :
#include <stdio.h>
int main() {
int a, b, c;
// Saisie des trois nombres par l'utilisateur
printf("Entrez le premier nombre : ");
scanf("%d", &a);
printf("Entrez le deuxième nombre : ");
scanf("%d", &b);
printf("Entrez le troisième nombre : ");
scanf("%d", &c);
// Recherche du maximum entre les trois nombres
int maximum = a; // Supposons que le premier nombre est le maximum
if (b > maximum) {
maximum = b;
}
if (c > maximum) {
maximum = c;
}
// Affichage du maximum
printf("Le maximum entre les trois nombres est : %d\n", maximum);
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur d'entrer trois nombres, puis nous les stockons dans les variables a, b et c. Ensuite, nous comparons les trois nombres pour trouver le maximum en utilisant des instructions conditionnelles (if). La variable maximum est initialisée avec la valeur du premier nombre, puis nous la mettons à jour si un nombre plus grand est trouvé. Enfin, nous affichons le maximum à l'aide de la fonction printf().

EXERCICE 3 :

Ecrire un programme en C qui lit les notes de deux contrôles d’une matière donnée, puis calcule et affiche la moyenne obtenue. si la moyenne est supérieure ou égale à 10, il affichera davantage le message de congratulation suivant : Félicitations ! Vous avez validé cette matière.
EXEMPLE D'EXECUTION :
Entrez la note du premier contrôle : 12.5
Entrez la note du deuxième contrôle : 14.8
La moyenne obtenue est : 13.65
Félicitations ! Vous avez validé cette matière.
SOLUTION :
#include <stdio.h>
int main() {
float a, b, moyenne;
// Saisie des notes des deux contrôles
printf("Entrez la note du premier contrôle : ");
scanf("%f", &a);
printf("Entrez la note du deuxième contrôle : ");
scanf("%f", &b);
// Calcul de la moyenne
moyenne = (a + b) / 2;
// Affichage de la moyenne
printf("La moyenne obtenue est : %.2f\n", moyenne);
// Vérification de la validation de la matière
if (moyenne >= 10) {
printf("Félicitations ! Vous avez validé cette matière.\n");
}
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur de saisir les notes des deux contrôles, puis nous les stockons dans les variables note1 et note2. Ensuite, nous calculons la moyenne en additionnant les deux notes et en divisant le résultat par 2. La moyenne est stockée dans la variable moyenne. Enfin, nous affichons la moyenne à l'aide de la fonction printf() avec le format "%.2f" pour afficher deux décimales, et nous affichons un message de félicitations si la moyenne est supérieure ou égale à 10.

EXERCICE 4 :

Écrivez un programme en C pour entrer le numéro du jour de la semaine (1-7) et affichez le nom du jour de la semaine correspondant en utilisant if-else.

#include <stdio.h>
int main() {

if (numeroJour == 1) {
printf("Lundi\n");
}
else if (numeroJour == 2) {
printf("Mardi\n");
}
else if (numeroJour == 3) {
printf("Mercredi\n");
}
else if (numeroJour == 4) {
printf("Jeudi\n");
}
else if (numeroJour == 5) {
printf("Vendredi\n");
}
else if (numeroJour == 6) {
printf("Samedi\n");
}
else if (numeroJour == 7) {
printf("Dimanche\n");
}
else {
printf("Numéro de jour invalide\n");
}
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur d'entrer le numéro du jour de la semaine, puis nous stockons cette valeur dans la variable numeroJour. Ensuite, nous utilisons une structure de contrôle if-else pour vérifier le numéro du jour et afficher le nom correspondant à l'aide de la fonction printf(). Si le numéro du jour n'est pas compris entre 1 et 7, nous affichons un message d'erreur.

EXERCICE 5 :

Ecrivez un programme en C qui demande à l'utilisateur de saisir le prix de fabrication et le prix de vente d'un produit. Ensuite, le programme vérifie s'il y a un profit ou une perte en comparant les deux prix.
EXEMPLE D'EXECUTION :
Entrez le prix de fabrication du produit : 50.00
Entrez le prix de vente du produit : 80.00
Profit : 30.00
SOLUTION :
#include <stdio.h>
int main() {
float prixFabrication, prixVente;
// Saisie du prix de fabrication
printf("Entrez le prix de fabrication du produit : ");
scanf("%f", &prixFabrication);
// Saisie du prix de vente
printf("Entrez le prix de vente du produit : ");
scanf("%f", &prixVente);
// Calcul du profit ou de la perte
float difference = prixVente - prixFabrication;
// Vérification et affichage du résultat
if (difference > 0) {
printf("Profit : %.2f\n", difference);
}
else if (difference < 0) {
printf("Perte : %.2f\n", -difference);
}
else {
printf("Pas de profit ni de perte\n");
}
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur de saisir le prix de fabrication et le prix de vente du produit, puis nous stockons ces valeurs dans les variables prixFabrication et prixVente. Ensuite, nous calculons la différence entre le prix de vente et le prix de fabrication à l'aide de l'opération prixVente - prixFabrication. Si cette différence est positive, cela signifie qu'il y a un profit, et nous l'affichons avec le message "Profit : " suivi du montant. Si la différence est négative, cela signifie qu'il y a une perte, et nous affichons le montant positif avec le message "Perte : ". Si la différence est nulle, cela signifie qu'il n'y a ni profit ni perte, et nous affichons le message "Pas de profit ni de perte".

EXERCICE 6:

le programme en C qui demande à l'utilisateur d'entrer un score numérique entre 0 et 100, puis qui classe la note en fonction des conditions suivantes :

  • Si le score est entre 80 et 89 (inclus), affichez "B".
  • Si le score est entre 70 et 79 (inclus), affichez "C".
  • Si le score est entre 60 et 69 (inclus), affichez "D".

Entrez le score : 85
B

#include <stdio.h>

printf("B\n");
}
else if (score >= 70 && score <= 79) {
printf("C\n");
}
else if (score >= 60 && score <= 69) {
printf("D\n");
}
else {
printf("F\n");
}
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur de saisir le score, puis nous stockons cette valeur dans la variable score. Ensuite, nous utilisons une série d'instructions if et else if pour classer la note en fonction du score. Si le score est supérieur ou égal à 90, nous affichons "A". Si le score est entre 80 et 89, nous affichons "B". Si le score est entre 70 et 79, nous affichons "C". Si le score est entre 60 et 69, nous affichons "D". Si le score est inférieur à 60, nous affichons "F".

EXERCICE 7:

Ecrivez un programme en C pour inventer une calculatrice qui exécute des opérations arithmétiques de base(additionn, soustration, multiplication ou division) en utilisant if else.
EXEMPLE D'EXECUTION :
Entrez l'opérateur (+, -, *, /) : *
Entrez le premier nombre : 5.2
Entrez le deuxième nombre : 3.8
Le résultat est : 19.76
SOLUTION :
#include <stdio.h>
int main() {
char operateur;
double nombre1, nombre2, resultat;
// Saisie de l'opération et des nombres
printf("Entrez l'opérateur (+, -, *, /) : ");
scanf(" %c", &operateur);
printf("Entrez le premier nombre : ");
scanf("%lf", &nombre1);
printf("Entrez le deuxième nombre : ");
scanf("%lf", &nombre2);
// Calcul et affichage du résultat
if (operateur == '+') {
resultat = nombre1 + nombre2;
}
else if (operateur == '-') {
resultat = nombre1 - nombre2;
}
else if (operateur == '*') {
resultat = nombre1 * nombre2;
}
else if (operateur == '/') {
if (nombre2 != 0) {
resultat = nombre1 / nombre2;
}
else {
printf("Erreur : Division par zéro n'est pas autorisée.\n");
return 1; // Quitter le programme avec code d'erreur 1
}
}
else {
printf("Opérateur invalide.\n");
return 1; // Quitter le programme avec code d'erreur 1
}
printf("Le résultat est : %.2lf\n", resultat);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous demandons à l'utilisateur de saisir l'opérateur (+, -, *, /) suivi des deux nombres sur lesquels l'opération sera effectuée. Ensuite, nous utilisons des instructions if et else if pour déterminer quelle opération doit être effectuée en fonction de l'opérateur saisi. Si l'opérateur est invalide (autre que +, -, * ou /), nous affichons un message d'erreur. Si l'opération de division est choisie et que le deuxième nombre est égal à zéro, nous affichons également un message d'erreur pour éviter la division par zéro.

Notez que le programme utilise le format %lf pour la saisie de nombres réels (double) avec scanf et utilise %.2lf pour afficher le résultat avec deux décimales.

scanf("%d", &nombre2);
printf("Entrez le troisième entier : ");
scanf("%d", &nombre3);
// Vérification de l'ordre des nombres et échange si nécessaire
if (nombre1 > nombre2) {
int temp = nombre1;
nombre1 = nombre2;
nombre2 = temp;
}
if (nombre2 > nombre3) {
int temp = nombre2;
nombre2 = nombre3;
nombre3 = temp;
}
if (nombre1 > nombre2) {
int temp = nombre1;
nombre1 = nombre2;
nombre2 = temp;
}
// Affichage des nombres dans l'ordre croissant
printf("Les nombres dans l'ordre croissant sont : %d %d %d\n", nombre1, nombre2, nombre3);

EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur de saisir trois entiers. EXERCICE 10:

Écrivez un programme en C qui détermine si une année donnée est bissextile ou non.
Une année est bissextile si elle est divisible par 4, sauf si elle est également divisible par 100. Toutefois, les années qui sont divisibles par 400 sont considérées comme bissextiles.
EXEMPLE D'EXECUTION :
Entrez une année : 2020
2020 est une année bissextile.
SOLUTION :
#include <stdio.h>
int main() {
int annee;
printf("Entrez une année : ");
scanf("%d", &annee);
// Vérification si l'année est bissextile ou non
if ((annee % 4 == 0 && annee % 100 != 0) || annee % 400 == 0) {
printf("%d est une année bissextile.\n", annee);
}
else {
printf("%d n'est pas une année bissextile.\n", annee);
}
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur de saisir une année. Ensuite, nous utilisons une expression conditionnelle pour vérifier si l'année est bissextile ou non. La condition est que l'année doit être divisible par 4 mais pas par 100, sauf si elle est également divisible par 400. Si la condition est vraie, alors l'année est bissextile, sinon, elle ne l'est pas. Enfin, nous utilisons l'instruction printf() pour afficher le résultat.
