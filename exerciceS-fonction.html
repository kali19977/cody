EXERCICE 1 :

Écrivez une fonction appelée calculer_carre qui prend un nombre en entrée et renvoie le carré de ce nombre.
SOLUTION :
def calculer_carre():
n = int(input("Entrez un nombre entier : "))
carre = n ** 2
return carre
resultat = calculer_carre()
print(resultat)
EXPLICATION :
Tout d'abord,
La fonction calculer_carre est définie sans aucun argument, ce qui signifie qu'elle ne prend pas de paramètre explicite. Cela permet à l'utilisateur d'entrer un nombre lors de l'exécution de la fonction.
Ensuite,
À l'intérieur de la fonction, nous utilisons la fonction input pour afficher un message demandant à l'utilisateur d'entrer un nombre entier. L'entrée de l'utilisateur est récupérée sous forme de chaîne de caractères.
Nous utilisons ensuite la fonction int() pour convertir la chaîne de caractères en un nombre entier. Cela garantit que l'entrée de l'utilisateur est traitée comme un nombre plutôt que comme une simple chaîne de caractères.
La variable n contient maintenant le nombre entier saisi par l'utilisateur.
Nous calculons le carré de n en utilisant l'opérateur d'exponentiation **. Cela soulève n à la puissance 2, ce qui donne le carré de n.
Le résultat du calcul du carré est stocké dans la variable carre.
Enfin,
la fonction calculer_carre renvoie la valeur de carre, qui est le carré du nombre saisi par l'utilisateur.

EXERCICE 5 :

Écrire une fonction appelée "est_premier" qui prend un nombre en entrée et renvoie True s'il est premier, et False sinon.
SOLUTION :
def est_premier(nombre):
if nombre < 2:
return False
for i in range(2, int(nombre**0.5) + 1):
if nombre % i == 0:
return False
return True
nombre_utilisateur = int(input("Entrez un nombre entier : "))
resultat = est_premier(nombre_utilisateur)
print(resultat)
EXPLICATION :
  • La fonction "est_premier" est définie avec un seul argument, "nombre", qui représente le nombre entier donné en entrée.
  • Dans la fonction, nous vérifions d'abord si le nombre est inférieur à 2. Si c'est le cas, nous retournons False car les nombres inférieurs à 2 ne peuvent pas être premiers.
  • Ensuite, nous utilisons une boucle "for" pour parcourir tous les nombres de 2 jusqu'à la racine carrée du nombre (int(nombre**0.5) + 1). Cela nous permet de vérifier si le nombre est divisible par un autre nombre entre 2 et sa racine carrée.
  • Si le nombre est divisible par un autre nombre, cela signifie qu'il n'est pas premier et nous retournons False.
  • Si aucune division exacte n'est trouvée dans la boucle, cela signifie que le nombre est premier et nous retournons True.
  • En dehors de la fonction, nous demandons à l'utilisateur d'entrer un nombre entier à l'aide de la fonction "input" et nous convertissons cette valeur en entier en utilisant "int()". Nous stockons cette valeur dans la variable "nombre_utilisateur".
  • Nous appelons ensuite la fonction "est_premier" avec "nombre_utilisateur" comme argument pour vérifier si le nombre est premier.
  • Enfin, nous affichons le résultat à l'écran.

Écrivez une fonction appelée est_palindrome qui prend une chaîne de caractères en entrée et renvoie True si la chaîne est un palindrome, sinon renvoie False.
SOLUTION :
def est_palindrome(chaine):
chaine_inverse = chaine[::-1]
if chaine == chaine_inverse:
return True
else:
return False
ch=input("saisir une chaine")
resultat = est_palindrome(ch)
print(resultat)
EXPLICATION :
La fonction est_palindrome utilise la même technique que l'exercice précédent pour inverser la chaîne de caractères. Ensuite, elle compare la chaîne originale avec sa version inversée pour déterminer si elles sont identiques.

EXERCICE 10:

Définir et utilise une fonction Permuter qui permute les valeurs de deux variables réelles.
SOLUTION :
def Permuter(a, b):
temp = a
a = b
b = temp
# Demande à l'utilisateur d'entrer deux variables réelles
x = float(input("Entrez la valeur de x : "))
y = float(input("Entrez la valeur de y : "))
# Affiche les valeurs avant permutation
print("Avant permutation :")
print("x =", x)
print("y =", y)
# Appelle la fonction Permuter pour permuter les valeurs
Permuter(x, y)
# Affiche les valeurs après permutation
print("Après permutation :")
print("x =", x)
print("y =", y)
EXPLICATION :
  • La fonction "Permuter" est définie pour prendre deux variables réelles, "a" et "b", en arguments.
  • À l'intérieur de la fonction, nous utilisons une variable temporaire "temp" pour stocker la valeur de "a".
  • Nous assignons ensuite la valeur de "b" à "a" et la valeur de "temp" à "b", ce qui permute les valeurs des variables.
  • En dehors de la fonction, nous demandons à l'utilisateur d'entrer les valeurs de deux variables réelles, "x" et "y".
  • Avant d'appeler la fonction "Permuter", nous affichons les valeurs des variables "x" et "y" pour montrer leur état avant la permutation.
  • Ensuite, nous appelons la fonction "Permuter" en passant les variables "x" et "y" en arguments. Cela provoque la permutation de leurs valeurs.
  • Enfin, nous affichons à nouveau les valeurs des variables "x" et "y" pour montrer leur état après la permutation.
