EXERCICE 1 :

Ecrire un programme en langage C qui utilise des pointeurs pour lire deux entiers saisis par l'utilisateur et calculer leur somme.
SOLUTION :
#include <stdio.h>

int entier1, entier2;

scanf("%d", &entier1);
printf("Entrez le deuxième entier : ");
scanf("%d", &entier2);
int somme = entier1 + entier2;
printf("La somme des deux entiers est : %d\n", somme);
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons des pointeurs ptr1 et ptr2 pour stocker les adresses des variables entier1 et entier2. Ensuite, nous utilisons scanf avec les pointeurs pour lire les valeurs saisies par l'utilisateur et les stocker dans les variables correspondantes. Enfin, nous utilisons les opérateurs de déréférencement * pour accéder aux valeurs pointées par les pointeurs et calculer leur somme. Le résultat est ensuite affiché à l'écran.

Écrire un programme en C qui utilise des pointeurs pour trier un tableau d'entiers saisi par l'utilisateur dans l'ordre croissant.

#include <stdio.h>
void trierTableau(int *tableau, int taille) {
int *ptr1, *ptr2;
int temp;
for (int i = 0; i < taille - 1; i++) {
ptr1 = tableau;
for (int j = i + 1; j < taille; j++) {
ptr2 = tableau + j;
if (*ptr1 > *ptr2) {
temp = *ptr1;
*ptr1 = *ptr2;
*ptr2 = temp;
}
ptr1++;
}
}
}
int main() {
int taille;
printf("Entrez la taille du tableau : ");
scanf("%d", &taille);
int tableau[taille];
printf("Entrez les elements du tableau :\n");
for (int i = 0; i < taille; i++) {
printf("Element %d : ", i + 1);
scanf("%d", &tableau[i]);
}
trierTableau(tableau, taille);
printf("Tableau trie dans l'ordre croissant :\n");
for (int i = 0; i < taille; i++) {
printf("%d ", tableau[i]);
}
printf("\n");
return 0;
}
EXPLICATION :
  • La fonction trierTableau prend un pointeur vers un tableau d'entiers (int *tableau) et la taille du tableau (int taille) en paramètres.
  • Nous déclarons deux pointeurs ptr1 et ptr2 pour parcourir le tableau et effectuer les comparaisons.
  • À l'aide de deux boucles for, nous comparons chaque élément du tableau avec tous les éléments qui le suivent.
  • Si un élément est trouvé plus grand que celui qui suit, nous échangeons leur position en utilisant les pointeurs.
  • Nous utilisons l'opérateur de déréférencement (*) pour accéder aux valeurs pointées par les pointeurs.
  • Dans la fonction main, nous demandons à l'utilisateur de saisir la taille du tableau.
  • Nous déclarons un tableau tableau de la taille saisie par l'utilisateur.

EXERCICE 10:

Ecrire un programme C qui utilise des pointeurs pour trouver l'élément maximum dans un tableau d'entiers saisi par l'utilisateur.
SOLUTION :
#include <stdio.h>
int findMax(int *array, int size) {
int max = *array; // Initialize max with the first element of the array
for (int i = 1; i < size; i++) {
if (*(array + i) > max) {
max = *(array + i); // Update max if a larger element is found
}
}
return max;
}
int main() {
int size;
printf("Enter the size of the array: ");
scanf("%d", &size);
int array[size];
printf("Enter the elements of the array:\n");
for (int i = 0; i < size; i++) {
scanf("%d", &array[i]);
}
int max = findMax(array, size);
printf("The maximum element in the array is: %d\n", max);
return 0;
}
EXPLICATION :
  • Le programme demande à l'utilisateur d'entrer la taille du tableau.
  • Un tableau d'entiers, tableau, est déclaré avec la taille donnée par l'utilisateur.
  • L'utilisateur est invité à entrer les éléments du tableau.
  • La fonction trouverMaximum est définie pour trouver l'élément maximum dans le tableau en utilisant des pointeurs.
  • Un pointeur tableau est passé à la fonction, ainsi que la taille du tableau.
  • Une variable maximum est initialisée avec la valeur du premier élément du tableau.
  • Une boucle est utilisée pour parcourir le reste du tableau en utilisant des pointeurs.
  • Si l'élément pointé est plus grand que le maximum actuel, la valeur maximale est mise à jour.
  • Une fois le parcours terminé, la fonction renvoie le maximum trouvé.
  • Dans la fonction main, la fonction trouverMaximum est appelée en passant le tableau et sa taille, et le résultat est stocké dans la variable maximum.
  • Le maximum est ensuite affiché à l'écran.
