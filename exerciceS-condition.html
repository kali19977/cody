EXERCICE 1 :

Ecrire un programme python qui permet de determiner si un entier donné est pair ou impair.
EXEMPLE D'EXECUTION :
Entrez un entier : 7
7 est un nombre impair.
SOLUTION :
entier = int(input("Entrez un entier : "))
if entier % 2 == 0:
print(entier, "est un nombre pair.")
else:
print(entier, "est un nombre impair.")
EXPLICATION :
Tout d'abord,
entier = int(input("Entrez un entier : ")) : Cette ligne demande à l'utilisateur d'entrer un entier en affichant le message "Entrez un entier : ". L'entrée de l'utilisateur est capturée à l'aide de la fonction input(), puis elle est convertie en entier à l'aide de la fonction int() et assignée à la variable entier.
Ensuite,
if entier % 2 == 0: : Cette ligne vérifie si l'entier est divisible par 2 sans reste, c'est-à-dire s'il est pair. L'opérateur % calcule le reste de la division de entier par 2. Si le reste est égal à 0, cela signifie que l'entier est pair.
print(entier, "est un nombre pair.") : Si l'entier est pair, cette ligne affiche le message "est un nombre pair." en concaténant la valeur de entier avec le texte à l'aide de la fonction print().
else: : Si l'entier n'est pas pair, cette ligne est exécutée.
Enfin,
print(entier, "est un nombre impair.") : Cette ligne affiche le message "est un nombre impair." si l'entier est impair.

Ecrivez un programme python pour trouver un maximum entre trois nombres .
EXEMPLE D'EXECUTION :
Entrez le premier nombre : 2.5
Entrez le deuxième nombre : 2.5
Entrez le troisième nombre : 2.5

EXPLICATION :
Tout d'abord,
Les variables a, b et c sont utilisées pour stocker les trois nombres saisis par l'utilisateur à l'aide de la fonction input(). Les nombres sont convertis en nombres à virgule flottante (float) à l'aide de float().
Ensuite,
En utilisant des conditions if et des opérateurs de comparaison (>=), nous comparons les nombres a, b et c pour déterminer le maximum entre eux.
Si a est supérieur ou égal à b et a est supérieur ou égal à c, alors a est le maximum. Sinon, si b est supérieur ou égal à a et b est supérieur ou égal à c, alors b est le maximum. Sinon, c est le maximum.
Le maximum est stocké dans la variable maximum.
Enfin,
Le résultat est ensuite affiché à l'aide de la fonction print(), avec le message "Le maximum est : " suivi du maximum calculé.

Ensuite,
La moyenne est calculée en additionnant les deux notes (note1 + note2) et en les divisant par 2.
Le résultat est ensuite affiché à l'aide de la fonction print(), avec le message "La moyenne obtenue est :" suivi de la moyenne calculée.
Enfin,
En utilisant une condition if, on vérifie si la moyenne est supérieure ou égale à 10. Si c'est le cas, le message "Félicitations ! Vous avez validé cette matière." est affiché.

EXERCICE 6:

le programme qui demande à l'utilisateur d'entrer un score numérique entre 0 et 100, puis qui classe la note en fonction des conditions suivantes :

  • Si le score est entre 70 et 79 (inclus), affichez "C".
  • Si le score est entre 60 et 69 (inclus), affichez "D".
  • Si le score est inférieur à 60, le programme affiche "F".

Entrez le score : 72.5
C

score = float(input("Entrez le score : "))
if score >= 90:
print("A")
elif 80 <= score <= 89:
print("B")
elif 70 <= score <= 79:
print("C")
elif 60 <= score <= 69:
print("D")
else:
print("F")
EXPLICATION :
Tout d'abord,

Ensuite,
le programme utilise une structure conditionnelle if-elif-else pour classer la note en fonction du score.
La première condition if score >= 90 vérifie si le score est égal ou supérieur à 90. Si c'est le cas, le programme affiche "A".
Si la première condition n'est pas satisfaite, le programme passe à la condition suivante elif 80 <= score <= 89. Cette condition vérifie si le score est compris entre 80 et 89 inclus. Si c'est le cas, le programme affiche "B".
Si la deuxième condition n'est pas satisfaite, le programme passe à la condition suivante elif 70 <= score <= 79. Cette condition vérifie si le score est compris entre 70 et 79 inclus. Si c'est le cas, le programme affiche "C".
Si la troisième condition n'est pas satisfaite, le programme passe à la condition suivante elif 60 <= score <= 69. Cette condition vérifie si le score est compris entre 60 et 69 inclus. Si c'est le cas, le programme affiche "D".
Si aucune des conditions précédentes n'est satisfaite, le programme exécute le bloc else et affiche "F", ce qui signifie que le score est inférieur à 60.
Enfin,
Une fois que le programme a déterminé la note en fonction du score et affiché le résultat, il se termine.

if num2 != 0:

num1 = int(input("Entrez le premier entier : "))

if num2 <= num3:

max_num = num3
else:

max_num = num2

max_num = num3

mid_num = num3

Nous utilisons des conditions similaires dans les autres blocs elif pour comparer num2 avec num1 et num3, ainsi que num3 avec num1 et num2. Cela nous permet de trouver les valeurs minimale, médiane et maximale dans chaque cas.

Enfin, nous utilisons print() pour afficher les entiers dans l'ordre croissant en les séparant par des virgules. Les valeurs sont affichées en utilisant les variables min_num, mid_num et max_num dans cet ordre.

Enfin,
Sinon, si le nombre n'est pas un entier, le programme affiche un message indiquant que le nombre n'est pas un nombre entier.

EXERCICE 10:

Écrivez un programme qui détermine si une année donnée est bissextile ou non.
Une année est bissextile si elle est divisible par 4, sauf si elle est également divisible par 100. Toutefois, les années qui sont divisibles par 400 sont considérées comme bissextiles.
EXEMPLE D'EXECUTION :
Entrez une année : 2021
2021 n'est pas une année bissextile.
SOLUTION :
annee = int(input("Entrez une année : "))
if annee % 4 == 0:
if annee % 100 == 0:
if annee % 400 == 0:
print(annee, "est une année bissextile.")
else:
print(annee, "n'est pas une année bissextile.")
else:
print(annee, "est une année bissextile.")
else:
print(annee, "n'est pas une année bissextile.")
EXPLICATION :
Tout d'abord,
Nous utilisons la fonction input() pour demander à l'utilisateur d'entrer une année. L'année est ensuite convertie en type int à l'aide de int(input()) pour permettre les opérations arithmétiques.
Ensuite,
En utilisant des instructions conditionnelles if, nous vérifions si l'année est divisible par 4 en utilisant l'opérateur % pour vérifier si le reste de la division par 4 est égal à 0.
Si l'année est divisible par 4, nous vérifions ensuite si elle est divisible par 100. Si c'est le cas, nous vérifions si elle est également divisible par 400. Si elle l'est, alors c'est une année bissextile. Sinon, ce n'est pas une année bissextile.
Si l'année est divisible par 4 mais pas par 100, alors c'est une année bissextile.
Si l'année n'est pas divisible par 4, alors ce n'est pas une année bissextile.
Enfin,
En utilisant print(), nous affichons le résultat de la détermination de l'année bissextile ou non.
