EXERCICE 1 :

Ecrire un programme en C qui demande à l'utilisateur de saisir deux nombres, puis calcule et affiche leur somme.
EXEMPLE D'EXECUTION :
Entrez le premier nombre : 4.5
Entrez le deuxième nombre : 2.3
La somme des deux nombres est : 6.80
SOLUTION :
#include <stdio.h>
int main() {
float nombre1, nombre2, somme;
// Demander à l'utilisateur de saisir les deux nombres
printf("Entrez le premier nombre : ");
scanf("%f", &nombre1);
printf("Entrez le deuxième nombre : ");
scanf("%f", &nombre2);
// Calculer la somme des deux nombres
somme = nombre1 + nombre2;
// Afficher la somme
printf("La somme des deux nombres est : %.2f\n", somme);
return 0;
}
EXPLICATION :
Ce programme utilise la bibliothèque standard stdio.h pour l'entrée/sortie et utilise la fonction scanf pour récupérer les nombres saisis par l'utilisateur. La somme des deux nombres est calculée en ajoutant nombre1 et nombre2, puis elle est affichée à l'aide de la fonction printf.

EXERCICE 2 :

Ecrire un programme en C qui permet à l'utilisateur de saisir deux nombres réels, puis affiche leur produit .
EXEMPLE D'EXECUTION :
Entrez le premier nombre réel : 2.5
Entrez le deuxième nombre réel : 3.2
Le produit des deux nombres est : 8.000000
SOLUTION :
#include <stdio.h>
int main() {
float nombre1, nombre2, produit;
printf("Entrez le premier nombre réel : ");
scanf("%f", &nombre1);
printf("Entrez le deuxième nombre réel : ");
scanf("%f", &nombre2);
produit = nombre1 * nombre2;
printf("Le produit des deux nombres est : %f", produit);
return 0;
}
EXPLICATION :
Dans ce programme, nous utilisons la fonction printf pour afficher un message demandant à l'utilisateur de saisir les deux nombres réels. Ensuite, nous utilisons la fonction scanf pour lire les nombres saisis par l'utilisateur et les stocker dans les variables nombre1 et nombre2. Le produit des deux nombres est calculé en multipliant nombre1 par nombre2 et stocké dans la variable produit. Enfin, nous utilisons la fonction printf pour afficher le resultat.

EXERCICE 3 :

Ecrire un programme en C qui demande à l'utilisateur de saisir son nom et prénom, puis affiche le message "Welcome, prénom nom"
EXEMPLE D'EXECUTION :
Entrez votre nom : Smith
Entrez votre prénom : John
Welcome, John Smith
SOLUTION :
#include <stdio.h>
int main() {
char nom[50], prenom[50];
printf("Entrez votre nom : ");
scanf("%s", nom);
printf("Entrez votre prénom : ");
scanf("%s", prenom);
printf("Welcome, %s %s\n", prenom, nom);
return 0;
}
EXPLICATION :
Dans ce programme,
nous utilisons des tableaux de caractères (nom et prenom) pour stocker les noms saisis par l'utilisateur. La fonction scanf est utilisée pour lire les entrées de l'utilisateur et les stocker dans les tableaux correspondants. Ensuite, nous utilisons la fonction printf pour afficher le message de bienvenue en concaténant le prénom et le nom saisis.

nous calculons le périmètre en utilisant la formule 2 * (longueur + largeur).
Enfin,
nous utilisons la fonction printf pour afficher le résultat du calcul du périmètre.

EXERCICE 5 :

Écrivez un programme en C pour saisir le nombre de jours de l'utilisateur et convertissez-le en années, semaines et jours.
EXEMPLE D'EXECUTION :
Entrez le nombre de jours : 1000
Le nombre de jours correspond à : 2 année(s), 46 semaine(s) et 6 jour(s).
SOLUTION :
#include <stdio.h>
int main() {
int jours, annees, semaines, joursRestants;
printf("Entrez le nombre de jours : ");
scanf("%d", &jours);
// Calcul des années
annees = jours / 365;
// Calcul des jours restants
joursRestants = jours % 365;
// Calcul des semaines
semaines = joursRestants / 7;
// Calcul des jours restants
joursRestants = joursRestants % 7;
printf("Le nombre de jours correspond à : %d année(s), %d semaine(s) et %d jour(s).\n", annees, semaines, joursRestants);
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons une variable jours de type int pour stocker le nombre de jours saisi par l'utilisateur.
Ensuite,
nous effectuons les calculs nécessaires pour convertir ce nombre en années, semaines et jours. La division entière (/) est utilisée pour calculer le nombre d'années et de semaines, et l'opérateur modulo (%) est utilisé pour obtenir le nombre de jours restants après avoir calculé les années et les semaines.
Enfin,
nous utilisons la fonction printf pour afficher le résultat de la conversion en années, semaines et jours.

EXERCICE 6:

Ecrire un programme en C qui lit trois entiers, calcule puis affiche leur moyenne.
EXEMPLE D'EXECUTION :
Entrez le premier entier : 10
Entrez le deuxième entier : 15
Entrez le troisième entier : 20
La moyenne des trois entiers est : 15.00
SOLUTION :
#include <stdio.h>
int main() {
int entier1, entier2, entier3;
float moyenne;
printf("Entrez le premier entier : ");
scanf("%d", &entier1);
printf("Entrez le deuxième entier : ");
scanf("%d", &entier2);
printf("Entrez le troisième entier : ");
scanf("%d", &entier3);
moyenne = (entier1 + entier2 + entier3) / 3.0;
printf("La moyenne des trois entiers est : %.2f\n", moyenne);
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons trois variables entier1, entier2 et entier3 de type int pour stocker les entiers saisis par l'utilisateur.
Ensuite,
nous utilisons la formule (entier1 + entier2 + entier3) / 3.0 pour calculer la moyenne des trois entiers. La division par 3.0 est utilisée pour obtenir un résultat décimal (de type float) au lieu d'une division entière.
Enfin,
nous utilisons la fonction printf avec le format %.2f pour afficher la moyenne avec deux chiffres après la virgule.

EXERCICE 7:

Ecrire un programme en C qui lit deux nombres en entrée à partir de l'utilisateur, puis calcule et affiche leur puissance
EXEMPLE D'EXECUTION :
Entrez le premier nombre : 2.5
Entrez le deuxième nombre : 3.0
La puissance de 2.50 élevé à la 3.00 est : 15.63
SOLUTION :
#include <stdio.h>
#include <math.h>
int main() {
double nombre1, nombre2, puissance;
printf("Entrez le premier nombre : ");
scanf("%lf", &nombre1);
printf("Entrez le deuxième nombre : ");
scanf("%lf", &nombre2);
puissance = pow(nombre1, nombre2);
printf("La puissance de %.2lf élevé à la %.2lf est : %.2lf\n", nombre1, nombre2, puissance);
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction printf pour afficher les messages demandant à l'utilisateur de saisir les deux nombres. Les nombres sont ensuite saisis à l'aide de la fonction scanf et stockés dans les variables nombre1 et nombre2.
Ensuite,
nous utilisons la fonction pow de la bibliothèque mathématique (math.h) pour calculer la puissance du premier nombre élevé à la deuxième nombre. Le résultat est stocké dans la variable puissance.
Enfin,
nous utilisons à nouveau la fonction printf pour afficher le résultat de la puissance avec deux décimales grâce au formatage %.2lf.

EXERCICE 8:

Ecrivez un programme en C pour saisir un nombre et calculer sa racine carrée.
EXEMPLE D'EXECUTION :
Entrez un nombre : 25.0
La racine carrée de 25.00 est 5.00
SOLUTION :
#include <stdio.h>
#include <math.h>
int main() {
double nombre, racine_carree;
// Saisie du nombre
printf("Entrez un nombre : ");
scanf("%lf", &nombre);
// Calcul de la racine carrée
racine_carree = sqrt(nombre);
// Affichage du résultat
printf("La racine carrée de %.2lf est %.2lf\n", nombre, racine_carree);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous utilisons la fonction sqrt() de la bibliothèque mathématique <math.h> pour calculer la racine carrée d'un nombre. Nous utilisons la fonction scanf() pour demander à l'utilisateur d'entrer un nombre, puis nous stockons cette valeur dans la variable nombre . le programme utilise la fonction input() pour demander à l'utilisateur de saisir un nombre. L'entrée de l'utilisateur est stockée dans la variable nombre. Pour permettre les calculs avec des valeurs décimales, la valeur saisie est convertie en nombre réel (flottant) à l'aide de float(). Le programme utilise la fonction math.sqrt() pour calculer la racine carrée du nombre saisi. La fonction sqrt() prend le nombre en argument et renvoie sa racine carrée. Le résultat est stocké dans la variable racine_carree.
Enfin,
nous utilisons sqrt() pour calculer la racine carrée, et enfin, nous affichons le résultat à l'aide de printf().

Notez que %lf est utilisé dans scanf() pour lire un nombre à virgule flottante (double), et %.2lf est utilisé dans printf() pour afficher le résultat avec deux décimales après la virgule.

EXERCICE 9:

Écrivez un programme pour saisir deux angles de l'utilisateur et trouver le troisième angle du triangle.
a+b+c=180
EXEMPLE D'EXECUTION :
Entrez le premier angle : 30.0
Entrez le deuxième angle : 60.0
Le troisième angle du triangle est : 90.00 degrés
SOLUTION :
#include <stdio.h>
int main() {
float angle1, angle2, angle3;
// Saisie des deux angles connus
printf("Entrez le premier angle : ");
scanf("%f", &angle1);
printf("Entrez le deuxième angle : ");
scanf("%f", &angle2);
// Calcul du troisième angle
angle3 = 180 - (angle1 + angle2);
// Affichage du résultat
printf("Le troisième angle du triangle est : %.2f degrés\n", angle3);
return 0;
}
EXPLICATION :
Tout d'abord,
dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur d'entrer les deux angles connus du triangle, puis nous les stockons dans les variables angle1 et angle2. Ensuite, nous calculons le troisième angle en soustrayant la somme des deux angles connus de 180 (puisque la somme des angles d'un triangle est toujours égale à 180 degrés). Enfin, nous affichons le résultat à l'aide de la fonction printf().

EXERCICE 10:

Ecrivez un programme en C pour saisir le côté d'un triangle équilatéral et trouvez la surface du triangle donné.
EXEMPLE D'EXECUTION :
Entrez la longueur du cote du triangle : 5.0
La surface du triangle est : 10.83
SOLUTION :
#include <stdio.h>
#include <math.h>
int main() {
float cote, surface;
// Saisie de la longueur du côté du triangle
printf("Entrez la longueur du cote du triangle : ");
scanf("%f", &cote);
// Calcul de la surface du triangle
surface = (sqrt(3) / 4) * pow(cote, 2);
// Affichage du résultat
printf("La surface du triangle est : %.2f\n", surface);
return 0;
}
EXPLICATION :
Tout d'abord,
Dans ce programme, nous utilisons la fonction scanf() pour demander à l'utilisateur d'entrer la longueur du côté du triangle équilatéral, puis nous la stockons dans la variable cote. Ensuite, nous calculons la surface du triangle en utilisant la formule (sqrt(3) / 4) * cote^2, où sqrt(3) représente la racine carrée de 3 et pow(cote, 2) calcule le carré de la longueur du côté. Enfin, nous affichons le résultat à l'aide de la fonction printf().
